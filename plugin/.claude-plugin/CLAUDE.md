# claude-project-flow Plugin

Project workflow management plugin for Claude Code.

## Available Skills
- `/claude-project-flow:setup` — Configure project paths
- `/claude-project-flow:project-init` — Initialize project tracking
- `/claude-project-flow:feature-init` — Start a new feature
- `/claude-project-flow:feature-plan` — Create implementation plan
- `/claude-project-flow:feature-requirements` — Define requirements
- `/claude-project-flow:feature-list` — List features and status
- `/claude-project-flow:feature-close` — Archive/cancel a feature
- `/claude-project-flow:feature-merge` — Merge feature branch
- `/claude-project-flow:feature-docs` — Generate documentation
- `/claude-project-flow:discover-patterns` — Analyze patterns
- `/claude-project-flow:session-save` — Save session progress

## MCP Tools
All tools are prefixed with `mcp__project-flow__`:
- `feature_list`, `feature_get`, `feature_update`
- `knowledge_search`, `knowledge_index`
- `project_list`, `project_register`
- `settings_get`, `settings_update`
